# 🏠 Airbnb Clone

Hi, I’m Deepti 👋
This is my **major full stack project**, inspired by Airbnb. I built this project to understand how real-world applications work — from frontend UI to backend logic and database integration.

---

## 🚀 About the Project

This is a complete web application where users can explore listings, create their own properties, and interact with a booking system.
I tried to make it feel like a **real product**, not just a basic project.

---

## ✨ Features

* 🔐 User Authentication (Signup / Login / Logout)
* 🏡 Create, edit, and delete property listings
* 🔍 Search and filter listings
* 📍 Map integration for location display
* 📅 Booking functionality
* ⭐ Reviews and ratings system
* 🎨 Clean and responsive UI

---

## 🛠 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Other:** REST APIs, Map integration

---

## 📁 Project Structure

* `models/` → database schemas
* `routes/` → route handling
* `controllers/` → application logic
* `views/` → UI templates
* `public/` → static files
* `utils/` → helper functions
* `app.js` → main server
* `Screenshots` -> Project screenshots
